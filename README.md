# Image Uploader

A clean and responsive image uploader built with HTML, CSS, and JavaScript.

## Demo
Check out the live demo [here](https://giovannicaiazzo01.github.io/image-uploader/)!
## Features
- Drag and drop image upload
- Click to choose a file
- Instant image preview
- File type validation
- Simple and modern UI

## Technologies
- HTML
- CSS
- JavaScript

## Purpose
This project is a small front-end component for uploading and previewing images in a user-friendly way.
